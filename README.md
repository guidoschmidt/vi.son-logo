### Development
1. Put an mp3 file into `src/mp3/`
2. Update the `data-path` attribute in `src/index.html`, e.g.:

```html
    <div id="logo-canvas-wrapper"
         data-song="mp3/patterns.202006.mp3"></div>
```

3. `npm install`
4. `npm start`
